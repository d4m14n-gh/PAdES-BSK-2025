function encryptWithAES ( data,
aesKey )

Encrypts data using AES-256-ECB encryption.

This function encrypts a given string or buffer using a 256-bit AES key and the ECB (Electronic Codebook) mode of operation. The encrypted result is returned as a Buffer.

@function

Parameters
{string|Buffer}data - The data to be encrypted. Can be a UTF-8 string or a Buffer.
{Buffer}aesKey - A 256-bit AES key (32 bytes). Must be a valid Buffer of exactly 32 bytes.
Returns
{Buffer} The encrypted data as a Buffer.
Exceptions
{Error}If the AES key is invalid (not a Buffer of 32 bytes) or encryption fails.

◆ generateRSAKeys()

function generateRSAKeys ( diskPath,
userPIN )

Generates an RSA key pair, encrypts the private key, and saves the keys to disk.

This function creates a 4096-bit RSA key pair, encrypts the private key using a user-provided PIN, and saves both the public and encrypted private keys as files on the specified disk path.

@function

Parameters
{string}diskPath - The directory path where the RSA key files will be saved. Must be a valid and writable directory path.
{string}userPIN - A PIN provided by the user to encrypt the private key. Must be a non-empty string.
Returns
{Promise<Object>} A promise that resolves to an object containing:
  • publicKeyPath (string): The file path of the saved public key.
  • privateKeyPath (string): The file path of the saved encrypted private key.
Exceptions
{Error}If key generation, encryption, or file writing fails.

◆ getAvailableDrives()

function getAvailableDrives ( )

Retrieves a list of available logical drives on the system.

This function executes a shell command to query logical disks using wmic. The results are processed to extract drive names (e.g., "C:", "D:").

@function

Returns
{Promise<string[]>} A promise that resolves to an array of drive names. Each drive is represented as a string (e.g., "C:").
Exceptions
{Error}If there is an issue executing the shell command.

Hashes a given PIN using the SHA-256 algorithm.

This function takes a string PIN as input, validates it, and returns a hashed value using the sha256 hashing algorithm. The hashed value is a Buffer object.

@function

Parameters
{string}pin - The PIN to be hashed. It must be a non-empty string.
Returns

Exceptions
{Error}If the input PIN is invalid (not a string or empty).
